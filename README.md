# 🏡 Airbnb Performance Dashboard (Power BI)

 📊 Project Overview
The Airbnb Performance Dashboard is a comprehensive data visualization project built using Power BI to analyze Airbnb listings, pricing trends, customer ratings, and review insights across major global cities.

This dashboard transforms raw Airbnb data into interactive, visually engaging insights that help understand:
* Market performance
* Customer satisfaction
* Pricing dynamics
* City-wise comparisons

🎯 Objectives
* Analyze Airbnb listing performance across different cities
* Identify trends in pricing, occupancy, and reviews
* Evaluate customer satisfaction using ratings data
* Provide actionable insights for hosts, investors, and analysts

 📁 Dataset Information
* Source: Extracted from Kaggle Open AirBnb Data Set 
* Data includes:
  * City names (e.g., Paris, Rome, New York, Sydney, etc.)
  * Price per night
  * Number of reviews
  * Ratings (cleanliness, location, value, etc.)
  * Availability & listing counts
    
📌 Dashboard Pages & Features
🔹 1. Overview Page (Performance Summary)
  * City-wise comparison of Airbnb listings
  * Price distribution trends
  * Industry growth timeline visualization
  * KPI Cards:
      * Average Price
      * Total Listings
      * Total Reviews
  * Visual storytelling of market growth phases (e.g., "Industry Boom Time")

⭐ 2. Ratings Analysis Page
* Heatmap visualization for rating categories:
  * Cleanliness
  * Location
  * Value
  * Communication
* City-wise rating comparison
* Identifies top-performing cities based on customer satisfaction
* Highlights consistency in high-rated locations

📝 3. Reviews Analysis Page
* Top reviewer cities (e.g., Paris, Rome, New York, Sydney, Mexico City, Rio de Janeiro)
* Review distribution by city
* Insights into customer engagement and feedback trends
* Helps understand which locations attract the most user interaction

 📈 Key Insights
* Major cities like Paris, New York, and Sydney show high engagement and consistent ratings
* Emerging cities demonstrate strong growth potential in pricing and reviews
* Ratings across categories remain above average, indicating strong customer satisfaction
* Review-heavy cities correlate with higher listing performance

 🎨 Design & UI Highlights
* Clean and modern dashboard layout
* Consistent color palette (soft neutrals + highlight tones)
* Use of:
  * Line charts for trends
  * Heatmaps for ratings
  * Bar charts for comparisons
* Background visuals enhance storytelling without clutter

 🛠️ Tools & Technologies Used
* Power BI – Data visualization & dashboard creation
* Excel / CSV – Data preprocessing
* DAX (Data Analysis Expressions) – Calculations and measures
 
 📌 Use Cases
* 📊 Data Analyst Portfolio Project
* 🏢 Airbnb Market Analysis
* 💼 Business Intelligence Demonstration
* 🎓 Academic / College Project

 🚀 How to Use
1. Open the Power BI `.pbix` file
2. Interact with filters (city, time, ratings)
3. Explore different pages for detailed insights
4. Use visuals to derive actionable conclusions
